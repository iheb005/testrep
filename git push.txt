1/

2/
git add.       // add all files to given folder
git add server.js   // add specific file to given folder

3/
git status   // check/view the files which will be commited

4/
git commit -m 'message'    // commit message which adds the changes to local repo

5/
git remote add origin 'url-git-repo'      //url which is your remote repo to where your local content from your repo is pushed

6/ 
git push -u origin master/main   //push local content to github

pull cmd is used to allow others to view the changes (espicially in organizations)

4/ git push origin 'newbranch'  //push repo to github
5/ new branch added , click on compare & pull request
6/ open a pull request by clicking on "create pull request"
